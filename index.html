Note: Column layout is shared by everyone. Use ↑/↓ to reorder, the checkbox to show/hide a column, and Add column to create a custom field (Text, Date, or Dropdown) that appears in the table and the task form. Built-in columns can be hidden and renamed but not deleted. Saving requires your Firestore rules to allow the manager to write /config/columns (and everyone to read it).
